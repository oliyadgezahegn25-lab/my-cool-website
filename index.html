System Interactive Guide

1. Registration Panel (Left Sidebar)
Fill in item attributes including name, category, brand, model, units, price, initial stock, and spec catalog link. Submit to save directly to the Supabase database.
2. Switch Views (Equipment vs. Chemicals)
Use the top tabs above the directory table to toggle between standard equipment/supplies and chemical reagents (which include CAS & Purity specs).
3. Item Actions (Edit & Sell)
Each table row features an Edit button (to modify item attributes or catalogue links) and a Sell button (to deduct stock and log transactions).
4. Sales Log Menu & Audit Trail
Click Sales Log Menu at the top right to review client sales records. Click Audit Trail to review real-time activity logs.
